# Lab2_Mult Python Project

A minimal, empty Python project scaffold.

## Quick start

1. Create a virtual environment (Windows PowerShell):

   ```powershell
   py -m venv .venv; .\.venv\Scripts\python.exe -m pip install -U pip
   ```

2. Activate the environment:

   ```powershell
   .\.venv\Scripts\Activate.ps1
   ```

3. Install dependencies (none by default):

   ```powershell
   pip install -r requirements.txt
   ```

4. Add your code under `src/`.
